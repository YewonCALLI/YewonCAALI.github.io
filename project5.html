Star Instrument

Motive
I thought it would be nice to make constellations and make music using the characteristics of the figure.

Input 1 : 도형의 크기
도형이 커질수록 Input 1의 값이 작아진다.

Input 2 : 도형의 복잡도(오목, 볼록)
오목다각형에 가까울수록 Input 2의 값이 커진다.

Input 3 : 도형의 높낮이
도형이 높을수록 Input3의 값이 커진다.
