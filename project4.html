Envelope(포락선)

포락선 공식을 사용했습니다.
I used the envelope formula.

Process
Make a curve by connecting the four points (Neighboring four dots) above the envelope function. Make a pattern on the canvas by rotating (360 Degree) the envelope curve. The center point of the curve changes according to the position of the mouse, moves the mouse, and can create a pattern in which the envelope function is rotated.
